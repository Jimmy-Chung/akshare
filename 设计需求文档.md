# 全球市场行情看板 - UI设计需求文档

## 一、项目概述

**项目名称**：全球市场行情看板

**项目描述**：一个展示全球股票市场行情数据的大屏看板Web应用，用户可以快速了解全球主要市场涨跌情况。

**目标用户**：投资者、金融从业者、对市场行情感兴趣的用户

---

## 二、页面整体布局

页面从上到下分为四个区域：

```
┌──────────────────────────────────────────────────┐
│  【区域1】顶部轮播条                               │
│  全球指数轮播展示                                  │
├──────────────────────────────────────────────────┤
│  【区域2】市场切换标签                              │
│  A股 | 港股 | 美股                                 │
├──────────────────────────────────────────────────┤
│  【区域3】指数概览                                 │
│  指数卡片 + 分时走势图                             │
├──────────────────────────────────────────────────┤
│  【区域4】热力图                                   │
│  Treemap方块图，展示涨跌情况                        │
└──────────────────────────────────────────────────┘
```

---

## 三、各区域设计需求

### 区域1：顶部轮播条

**功能说明**：
- 自动轮播展示全球主要市场的指数涨跌情况
- 像新闻滚动条一样，从左向右或卡片切换形式循环播放

**展示内容**：
每个指数卡片包含：
- 指数名称（如"上证指数"、"道琼斯"、"日经225"）
- 当前点位（如 4112.90）
- 涨跌幅百分比（如 +1.25% 或 -0.83%）
- 涨跌额（如 +35.62 或 -10.50）

**需要轮播的指数列表**：

| 市场区域 | 指数名称 |
|----------|----------|
| 中国A股 | 上证指数、深证成指、创业板指、科创50 |
| 港股 | 恒生指数、恒生科技 |
| 美股 | 道琼斯、纳斯达克、标普500 |
| 日本 | 日经225 |
| 韩国 | 韩国综合指数 |
| 英国 | 富时100 |
| 德国 | DAX30 |
| 法国 | CAC40 |
| 印度 | 孟买指数 |
| 澳大利亚 | S&P200 |

共计约15个指数，循环轮播。

**颜色规范**：
- 涨：**红色**（建议 #E53935）
- 蒸：**绿色**（建议 #43A047）
- 平盘/无变化：灰色或中性色

**数字格式**：
- 涨跌幅显示百分比，保留两位小数
- 正数带"+"号，负数带"-"号
- 例如：+1.25%、-0.83%

**轮播行为**：
- 自动播放，每张卡片停留约3-5秒
- 鼠标悬停时可暂停轮播（可选）

---

### 区域2：市场切换标签

**功能说明**：
- 用户点击切换查看不同市场的详细数据
- 三个选项：**A股**、**港股**、**美股**
- 默认选中A股

**交互行为**：
- 点击某个标签后，下方区域3和区域4的内容随之切换
- 选中的标签需要有明显的高亮状态（背景色变化或下划线）

**视觉要求**：
- 标签样式清晰，易于点击
- 选中态与未选中态有明显区分
- 未选中态可用灰色弱化处理

---

### 区域3：指数概览

**功能说明**：
- 展示当前选中市场的主要指数涨跌情况
- 展示选中指数的当日分时走势图

#### 3.1 指数卡片区

每个市场展示4-5个主要指数：

**A股（默认）**：
- 上证指数、深证成指、创业板指、科创50、北证50

**港股**：
- 恒生指数、恒生科技、国企指数、红筹指数

**美股**：
- 道琼斯、纳斯达克、标普500、费城半导体

**卡片内容**：
- 指数名称
- 当前点位
- 涨跌幅（百分比）
- 涨跌额

**颜色规范**：
- 涨：红色
- 蒸：绿色

**排列方式**：
- 横向排列或网格布局，每个卡片大小一致

#### 3.2 分时走势图

**图表类型**：折线图（分时图）

**展示内容**：
- 展示选中指数当天的价格走势
- X轴为时间（如A股：9:30-15:00）
- Y轴为指数点位

**视觉要求**：
- 折线颜色：涨时用红色，跌时用绿色
- 基准线：昨日收盘价位置画一条横线（作为涨跌分界）
- 最新点位可标记高亮圆点
- 背景可填充渐变色：涨的区域红色背景，跌的区域绿色背景（可选）

**交互**（可选）：
- 点击上方某个指数卡片，走势图切换为该指数
- 鼠标悬停走势图，显示该时间点的具体数据

---

### 区域4：热力图

**功能说明**：
- 用方块大小和颜色直观展示涨跌情况
- 不同市场展示不同内容

#### 4.1 A股市场 → 板块热力图

**展示内容**：
- 每个方块代表一个行业/概念板块
- 方块颜色反映涨跌幅（红涨绿跌）
- 方块大小反映用户选择的指标大小

**方块内容**：
- 板块名称
- 涨跌幅百分比

**方块大小排序方式**（用户可切换）：

| 排序方式 | 说明 |
|----------|------|
| 按市值 | 板块总市值越大，方块越大 |
| 按涨跌幅 | 涨跌幅数值越大，方块越大 |
| 按净流入 | 主力净流入金额越大，方块越大 |

提供三个切换按钮：**市值** | **涨跌幅** | **净流入**

**颜色深度**：
- 涨幅越大，红色越深
- 蒸幅越大，绿色越深
- 涨跌幅接近0时，颜色较浅

**方块数量**：
展示主要板块，约50-80个方块

**布局方式**：
- Treemap（树状图）布局
- 方块紧凑排列，大方块在上或左，小方块填充剩余空间
- 类似股票看盘软件的热力图样式

**Hover交互**：

鼠标悬停某个板块方块时，显示悬浮信息卡片：

**悬浮卡片内容**：
- 板块名称
- 涨跌幅
- 净流入金额
- **该板块十大龙头股列表**（按市值从大到小排序）

龙头股列表每项显示：
- 股票名称
- 当前价格
- 涨跌幅
- 市值

#### 4.2 港股/美股市场 → 个股热力图

**展示内容**：
- 每个方块代表一只个股
- 方块颜色反映涨跌幅（红涨绿跌）
- 方块大小反映市值或涨跌幅

**方块内容**：
- 股票名称或代码
- 涨跌幅百分比

**方块大小排序方式**（用户可切换）：

| 排序方式 | 说明 |
|----------|------|
| 按市值 | 市值越大，方块越大 |
| 按涨跌幅 | 涨跌幅数值越大，方块越大 |

提供两个切换按钮：**市值** | **涨跌幅**

**方块数量**：
展示主要个股，约100-150只（按市值筛选主要股票）

**Hover交互**：

鼠标悬停某个股票方块时，显示悬浮信息卡片：

**悬浮卡片内容**：
- 股票名称
- 股票代码
- 当前价格
- 涨跌幅
- 涨跌额
- 市值

## 四、整体颜色规范

### 核心涨跌配色

| 状态 | 颜色 | 说明 |
|------|------|------|
| 涨 | 红色 #E53935 | 数值为正 |
| 蒸 | 绿色 #43A047 | 数值为负 |
| 平盘 | 灰色 #9E9E9E | 数值为0或接近0 |

### 热力图颜色梯度

涨跌幅度越大，颜色越深；幅度越小，颜色越浅。

建议设计一套渐变色：
- 深红 → 中红 → 浅红 → 灰色 → 浅绿 → 中绿 → 深绿

### 其他颜色

| 元素 | 建议颜色 |
|------|----------|
| 页面背景 | 深色背景（如 #1A1A2E）或浅色背景（设计师可决定） |
| 文字主色 | 根据背景选择，保证可读性 |
| 卡片背景 | 比页面背景稍浅或稍深，形成层次 |
| 边框/分割线 | 低对比度线条 |

---

## 五、数据刷新说明

**刷新频率**：每5分钟自动刷新一次所有数据

**刷新行为**：
- 刷新时显示短暂的加载状态（如微妙的闪烁或旋转图标）
- 数据更新后，界面平滑过渡显示新数据
- 无需用户手动点击刷新
